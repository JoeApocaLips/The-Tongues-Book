# The Tongues Book

> ⚠️ **Work in progress — under construction**

**[📄 Download PDF preview](https://github.com/JoeApocaLips/The-Tongues-Book/releases/download/version1.0/tongues_book.pdf)**

**The Tongues Book** is an automated pipeline for generating a 200-image book of AI-generated tongue photographs, published via Amazon KDP.

A parody of *[The Tongue Machine](https://ansgarallen.com/the-tongue-machine/)* by Ansgar Allen (Schism Press, March 2026).

> *In February 2026, Ansgar Allen issued a call for tongues. Email a photo. Full face or just the mouth. B&W. Each to appear for a fraction of a second.*
> *We did not email.*

Images and captions generated by [Grok](https://x.ai/) (xAI). Layout compiled with XeLaTeX. Published via Amazon KDP.

---

## Concept

Ansgar Allen made a video — a gallery of tongues, filmed close. The tongue as object. The tongue detached from speech, from meaning, from the machine that runs on it.

This project takes that literally: 200 tongues, no photographer, no subject, no session. A pipeline running in the dark, producing tongues on demand. Parameters sampled at random — pose, expression, framing, lighting, background, piercing, artifact — the machine decided. The author selected.

The AI model brought its own biases — aesthetic preferences, demographic defaults, cultural assumptions baked into training data. It hallucinated where it didn't know. It standardized where it should have varied. What remains is neither documentary nor fiction — it is the negotiated output of a machine that never saw a tongue, and a process that never had a subject.

## Pipeline

```
1. generate.bat      ->  tongues/*.png + *.jpg + *.json
2. reprocess.bat     ->  reprocess PNG -> JPG if needed (B&W, 2400px, 300dpi)
3. contact.bat       ->  contact.html — visual selection grid (click = select)
4. select.bat        ->  apply to_add.txt / to_remove.txt -> tongues/for_pdf/
5. captions.bat      ->  tongues/for_pdf/*.txt  (Boiffard-style captions via Grok)
6. find_bad.bat      ->  find captions matching bad_patterns.txt -> review file
7. all_captions.bat  ->  dump all captions to review file
8. del_bad.bat       ->  delete captions marked '-' in review file, regenerate
9. make_pdf.bat      ->  tongues_book.tex -> xelatex -> tongues_book.pdf
```

## Usage

### Requirements

```bash
pip install pillow numpy requests
# Python 3.11+ required (tomllib in stdlib)
```

- XeLaTeX (MiKTeX or TeX Live)
- Iosevka Slab Extended font installed
- xAI API key

### Environment

```bash
set XAI_API_KEY=your_key_here     # Windows
export XAI_API_KEY=your_key_here  # Linux/macOS
```

### Generate images

```bash
python The-Tongues-Book.py 10 --mode classic
python The-Tongues-Book.py 10 --mode wild
```

### Visual selection (contact sheet)

```
1. contact.bat          -> generates contact.html, opens in browser
2. click images         -> grey = unselected, green = selected, red = to remove
3. "Export & Apply"     -> downloads to_add.txt and/or to_remove.txt
4. select.bat           -> applies selection (looks in project dir then Downloads/)
```

### Caption workflow

```
captions.bat            -> generate missing captions
find_bad_captions.bat   -> find patterns, open review in Notepad
all_captions.bat        -> review all captions in Notepad
del_bad_captions.bat    -> delete marked (-) captions, regenerate
```

### Build PDF

```bash
python The-Tongues-Book.py --pdf
xelatex tongues_book.tex
```

## Modes

| Mode | Description |

| `classic` | smartphone selfies, bathroom mirrors, polaroids, webcam, documentary, mugshots, 35mm, photo booth |
| `wild` | fine art, surrealist, medical, forensic, neon, endoscope, night vision, infrared, thermal |

## Parameters

All generation parameters in `params.toml` — edit freely without touching the Python source. Lists + weights for: poses, expressions, framings, photo types, capture devices, lighting, backgrounds, artifacts, piercings, props, hand details, accessories, genders, ethnicities.

Compatibility rules prevent incoherent combinations (analog photo types exclude digital artifacts, mugshots restricted to appropriate backgrounds, etc.).

---

## Plate Styles

The LaTeX template auto-detects image properties and selects the appropriate style:

| Style | Condition |
|-------|-----------|
| `platedefault` | portrait, no white border |
| `plateblack` | portrait, white border (polaroid etc.) → black background |
| `platerotated` | landscape, no white border → rotated 90° |
| `platerotatedblack` | landscape, white border → rotated + black background |

## Caption Style

Captions follow the aesthetic of Jacques-André Boiffard's captions for *Documents* magazine (1929). A fragment, lowercase, sparse punctuation. Clinical or disturbing. The caption does not describe — it displaces.

> *the tongue that tasted the bleach first*
> *stolen from her mouth mid-kiss, glossy with someone else's gin*
> *the taste of five strangers, still clinging*

Bad patterns filtered via `bad_patterns.txt` (regex, one per line, editable).
System prompt fully editable in `captions_system_prompt.txt`.

## KDP Specs

- Format: 7×10 inches
- Interior: B&W, 300dpi (upscaled from Grok output via LANCZOS)
- Font: Iosevka Slab Extended 10pt
- ~200 pages (one image per page)
- No page numbers
